# 🤖 AIoT Pro Trading Terminal — Strategy Fatigue Detection System

[![Python](https://img.shields.io/badge/Python-3.10+-blue?logo=python)](https://python.org)
[![Flask](https://img.shields.io/badge/Flask-3.x-black?logo=flask)](https://flask.palletsprojects.com)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

A **production-grade AIoT web application** that detects strategy fatigue in algorithmic trading. Combines a Flask/Socket.IO backend, a dual-AI anomaly detection engine, real-time Finnhub market data, and a Node-RED IoT monitoring dashboard into a single, professional trading terminal.

---

## ✨ Features

| Category | Details |
|---|---|
| **Live Data Streaming** | WebSocket-based real-time tick delivery for zero-lag chart updates |
| **Universal Market Support** | US Stocks (AAPL, TSLA), Indian Stocks (RELIANCE.NS), Crypto (BTC-USD), Forex |
| **Dual AI Fatigue Engine** | MLP Autoencoder + Isolation Forest — detects strategy exhaustion via RSI, MACD, Bollinger Bands |
| **Premium UI** | Dark glassmorphism terminal with Lightweight Charts, scrolling ticker tape, blinking price flashes |
| **Paper Trading Simulator** | Portfolio & trade history with ₹1,00,000 starting balance |
| **IoT Dashboard** | Node-RED integration for real-time fatigue score monitoring |
| **Auth System** | Secure user registration & login with session management |

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/RithishSuresh/Fatigue-Detection-App.git
cd Fatigue-Detection-App
```

### 2. Create and activate a virtual environment
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Train the AI models
```bash
python train_dummy_models.py
```

### 5. Run the application
```bash
python app.py
```

Open **http://127.0.0.1:5000** in your browser and log in with:
- **Username**: `admin`
- **Password**: `admin123`

---

## 📡 Node-RED Integration

1. Install [Node-RED](https://nodered.org/) locally.
2. Import `nodered_flow.json` into your Node-RED instance.
3. Open the dashboard at **http://127.0.0.1:1880/ui** to monitor live fatigue scores.

## 📁 Project Structure

```
├── app.py                  # Main Flask application & Socket.IO server
├── ai_engine.py            # Dual AI engine (Autoencoder + Isolation Forest)
├── market_data.py          # Finnhub real-time data fetching & caching
├── database.py             # SQLite ORM helpers
├── train_dummy_models.py   # Script to generate pre-trained model weights
├── requirements.txt        # Python dependencies
├── nodered_flow.json       # Node-RED IoT flow export
├── templates/              # Jinja2 HTML templates (dashboard, login, register)
├── static/                 # CSS, JS, and image assets
└── models/                 # Saved AI model weights (generated by train script)
```

## 🛠️ Tech Stack

- **Backend**: Python 3.10+, Flask, Flask-SocketIO, Flask-Login
- **AI/ML**: scikit-learn (Isolation Forest), TensorFlow/Keras (MLP Autoencoder)
- **Market Data**: Finnhub API, yfinance (fallback)
- **Frontend**: Lightweight Charts, Socket.IO client, Glassmorphism CSS
- **IoT Layer**: Node-RED
- **Database**: SQLite
